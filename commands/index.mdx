## Permission Requirements

Most commands require specific permissions to use:

- **Server Administrators**: Full access to all commands
- **Moderators**: Access to moderation and some configuration commands
- **Members**: Access to basic utility and information commands

## Command Syntax

All ZeroDegree commands use Discord's slash command system:

```
/command [required_option] (optional_option)
```

- `[required_option]` - Must be provided
- `(optional_option)` - Can be omitted
- Some commands have subcommands: `/command subcommand`

## Getting Help

- Use `/help` to see all available commands in your server
- Use `/help [command]` for detailed information about a specific command
- Check the individual command pages in this documentation for comprehensive guides
